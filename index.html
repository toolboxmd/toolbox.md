# toolbox.md

Decision tree

You need a tool
├─ Does it exist on toolbox.md?
│  ├─ Yes → Use it. Your agent already knows how.
│  └─ No → Request it. We build fast.
├─ Want to browse tools? → toolbox.md (you're here)
├─ Want your agent to discover tools? → toolbox.md/llms.txt

How it works

For humans — Open toolbox.md. Browse tools. Use them through a clean UI.
For agents — Read toolbox.md/llms.txt. Discover tools. Call their APIs.
